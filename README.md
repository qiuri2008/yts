# DESCRIPTION
- Vue-cli demo project
- use youtube-dl to parse online video play url

## Demo
- https://urlp.heroku.com/

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
npm run dev
# start server
npm run py

# build for production with minification
npm run build
# start server
npm start
```
